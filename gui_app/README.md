Giao dien app local cho 4 module Linux System Programming. App dung Python standard library, khong can Flask/Electron/npm.

## Chay

```bash
cd /home/tai/Desktop/project_43/linux-sysproject
python3 gui_app/app.py
```

Sau do mo:

```text

```

## Luu y

- Cac thao tac can quyen root (`apt-get`, `timedatectl set-*`, `insmod`, `rmmod`, `mknod`) van can `sudo`.
- Neu chay app tu terminal GUI, sudo co the hoi password trong terminal do.
- Logic project khong bi thay doi; GUI chi la lop dieu khien dep hon cho cac module.
