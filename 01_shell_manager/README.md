# Module 1 - Shell Script Manager

Menu terminal bang Bash thuan, dung ANSI colors va box drawing chars.

## Chay

```bash
make run
```

```bash

```

## Chuc nang

- Quan ly file/thu muc: tao, xoa co confirm, copy/move, tim theo ten/noi dung.
- Cron scheduler: xem, them, xoa job theo so thu tu.
- Time manager: xem gio, timezone, NTP status, dat timezone/gio bang `timedatectl`.
- Package manager: tim, cai, go package voi confirm va spinner.

Mot so thao tac he thong can `sudo`.
